Sample D: Bideo.js, para hacer fondos de vídeo que cubran un DIV ( docs )

La gracia de Bideo.js es hacer un background de fondo a todo el ancho x alto de la página (o de un contendor) y el vídeo cubre el contenedor. En IE (9, 10, 11) no funciona la propiedad object-fit: cover; necesaria para hacer esto por CSS puro.
